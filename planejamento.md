# Projeto 1 – Meu Primeiro Portfólio

## Objetivo

Criar uma página simples com:
- Seu nome;
- Uma foto;
- Uma descrição sobre você;
- Uma lista de coisas que você gosta.

## Teoria rápida

- **HTML** define a estrutura da página (títulos, parágrafos, imagens, listas, links).
- **CSS** controla a parte visual (cores, fontes, espaçamentos, posicionamento).
- Os dois arquivos principais deste projeto são `index.html` e `style.css`.

## Planejamento passo a passo

1. Criar um repositório no GitHub chamado `meu-primeiro-portfolio` (sem espaços).
2. Criar no computador uma pasta com o mesmo nome e inicializar o repositório local (opcional).
3. Abrir essa pasta no Cursor por `File → Open Folder`.
4. Dentro da pasta, criar os arquivos `index.html` e `style.css`.
5. Montar o HTML com as seções principais:
   - Cabeçalho com seu nome e mensagem de apresentação;
   - Bloco “Sobre mim” com uma breve biografia;
   - Lista das coisas que você gosta;
   - Rodapé com créditos ou mensagem final.
6. Adicionar uma foto no HTML (pode ser usando uma imagem local ou URL).
7. Estilizar cada seção no `style.css`, aplicando cores, tipografia e espaçamento consistentes.
8. Revisar no navegador e ajustar responsividade básica (fontes legíveis em telas pequenas).
9. Atualizar este documento e o `README.md` quando mudar o escopo ou adicionar novas etapas.

## Próximos passos sugeridos

- Adicionar mais detalhes e links (por exemplo, redes sociais ou projetos).
- Criar uma versão publicada (GitHub Pages) para compartilhar o portfólio.
- Experimentar com variáveis CSS adicionais e animações simples.
