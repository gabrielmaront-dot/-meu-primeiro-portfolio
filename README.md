# Meu Primeiro Portfólio

Este repositório contém o primeiro exercício de criação de portfólio pessoal utilizando HTML e CSS. O objetivo é praticar a construção da estrutura da página e a aplicação de estilos básicos.

## Conteúdo

- `index.html`: estrutura da página com informações sobre o autor.
- `style.css`: estilos e cores aplicados ao layout.
- `planejamento.md`: documentação com objetivo, teoria rápida e plano de execução.

## Como visualizar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente no navegador ou utilize uma extensão de live server para visualizar as alterações em tempo real.

## Personalização sugerida

- Substitua `[SEU NOME]` pelo seu nome real.
- Adicione sua foto e links para redes sociais ou outros projetos.
- Ajuste cores, fontes e espaçamentos para refletir seu estilo pessoal.
